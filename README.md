# Commander
Simple Android Shell Command Executor
For daily use, we recommend **compiling it yourself** rather than using the pre-compiled binary APKs from the Releases; please refer to the "How to package" section.

## How to package (compile)
Packaging environment requirement: __iApp V3.0.1039__
Pay special attention! Does your iApp include the Material library? (This applies if you haven't upgraded.)

Download the Commander .iApp project (non-binary) from the Release, then import it into iApp via the system Share function.
<p align="center">
  <img src="github/1.png" width="180"/>
</p>
Click on the Commander project to view details.
<p align="center">
  <img src="github/2.png" width="180"/>
</p>
Then click "Package and Publish" to automatically compile and install.
